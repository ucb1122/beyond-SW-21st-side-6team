<Artist 정보에 필요한 변수>
- 고유ID :  artistId
- 밴드 이름 : artistName
- 장르 : genre
- 데뷔년도 : debutYear
- 소개 : description
- 대표곡 : hitSong
- 멤버 수 : memberCount
- 공식 홈페이지 : homepageUrl
- SNS 링크 : snsLink

<Concert 정보에 필요한 변수>
- 공연ID : concertId
- 공연 제목 : title
- 공연 날짜 : ConcertDate
- 공연 장소 : location
- 공연 소개 : description
- 출연 밴드 ID : artistId

<Seat 정보에 필요한 변수>
- 좌석 ID : seatId
- 공연 ID : concertId
- 좌석 번호	: seatNumber
- 좌석 가격	: price
- 예약 여부	: isReserved

----

<User 정보에 필요한 변수>
- 회원ID : userId
- 회원 비밀번호 : pwd
- 회원 이름 : userName	
- 예치금 : deposit

<예약 취소>
- 예약 ID :	reservationId	
- 취소 여부 : isCancelled
- 취소 일시	: cancelledAt	
- 환불 여부	: isRefunded
- 환불 금액	: refundPrice

- 예약 테이블과 예약 좌석 테이블을 분리

<예약 테이블>
- 예매 고유번호 : reservation_id (pk)
- 예매자 : userId
- 공연 : conertId
- 총 결제금액 : totalPrice
- 예매 일시 : reservationTime

<예약 좌석 테이블>
- 좌석 저장용 ID : seatId 
- 애매 고유번호 : reservationId (fk)
  
----

<Refund 정보에 필요한 변수>
- 공연ID : concertId
- 공연 날짜 : ConcertDate
- 좌석 ID : seatId
- 가격 : price
- 총 결제금액 : totalPrice
- 환불가능여부 : isRefundable
- 환불 요청 시간	refundRequestedAt	
- 환불 처리 완료 시간	refundProcessedAt
- 환불 사유	refundReason
- 예매 번호	reservationId

<payment 정보에 필요한 변수>
- 결제 ID : paymentId	
- 예매 ID : reservationId
- 가격 : price
- 총 결제금액 : totalPrice
- 결제 방법	: paymentMethod
- 결제 상태	: paymentStatus
  - PAID
  - CANCELLED
  - REFUNDED
- 결제 일시	paymentDate
- 공연ID : conertId
- 가격(좌석 1개당 금액) : price
- 총 결제금액 : totalPrice
- 예매 일시 : reservationTime
